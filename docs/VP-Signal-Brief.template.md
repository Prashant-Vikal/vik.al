## How to use this
- Each competency lists: what the interviewer is testing, the evidence (with IDs),
  and a story-ready version for verbal answers.
- Lean hardest on the competencies your decisions actually support. A solo build
  evidences judgment, prioritization, and tradeoffs strongly; it evidences
  people-leadership weakly — don't overclaim the thin ones.
- Pull `Outcome / impact` from the log wherever it's filled in; "decided X → it
  produced Y" beats the decision alone every time.
